Лабораторная работа №4: Single Page Application (Angular + EJB)

Стек: Angular (v2+), Java EE (EJB), REST API, JPA, СУБД HSQLDB.

Что реализовано:

— Архитектура: Полноценное разделение на Front-end (Angular) и Back-end (Java EE). Взаимодействие реализовано через REST API.

— Back-end (Java EE):

\* Бизнес-логика вынесена в EJB (Enterprise JavaBeans).

\* Реализована система аутентификации. Информация о пользователях хранится в БД, пароли защищены хэшированием.

\* Работа с базой данных HSQLDB через JPA.

— Front-end (Angular):

\* Реализована сложная валидация форм.

\* Адаптивная верстка: Интерфейс полностью адаптирован под три режима (Десктоп ≥ 1211px, Планшет ≥ 667px, Мобильный < 667px).

— Интерактив: Динамическая отрисовка области. Реализован механизм отправки координат по клику на график и автоматическое обновление точек.

— Безопасность: Реализован контроль доступа (Auth Guard) — неавторизованные пользователи не могут попасть на основную страницу.
