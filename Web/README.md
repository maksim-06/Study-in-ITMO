Веб-программирование (Java Enterprise \& Fullstack)

В этом разделе собраны мои лабораторные работы по курсу «Веб-программирование». Проекты демонстрируют эволюцию веб-технологий: от ручной настройки HTTP-серверов до разработки современных SPA-приложений.

🚀 Стек технологий:

Frontend: Angular (2+), TypeScript, JSF (PrimeFaces), JavaScript (AJAX).

Backend: Java EE (EJB, Servlets), REST API, FastCGI.

Data: JPA (ORM), БД (HSQLDB, PostgreSQL, Apache Derby).

Servers: WildFly.



📂 Обзор проектов:



Лабораторная работа №1: FastCGI и основы Web

Суть: Разработка FastCGI-сервера на Java для расчетов координат.

Технологии: Java FastCGI SDK, Native AJAX, Apache httpd.

Особенности: Блочная верстка, валидация на стороне клиента, конфигурация веб-сервера.



Лабораторная работа №2: Сервлеты и паттерн MVC

Суть: Приложение для проверки попадания точки в область, реализованное по архитектуре MVC.

Технологии: Java Servlets, JSP, HTTP-сессии.

Особенности: Использование ControllerServlet как единой точки входа, интерактивный кликабельный график, хранение истории в сессии.



Лабораторная работа №3: Компонентный подход с JSF

Суть: Перенос логики на фреймворк JSF с интеграцией базы данных.

Технологии: JSF, Managed Beans, JPA, PrimeFaces.

Особенности: Facelets-шаблоны, автоматическое обновление данных через AJAX, динамическая отрисовка области в зависимости от радиуса R.



Лабораторная работа №4: Fullstack SPA (Angular + EJB)

Суть: Современное веб-приложение с разделенным фронтендом и бэкендом.

Frontend: Angular, адаптивная верстка (Mobile/Tablet/Desktop).

Backend: EJB (бизнес-логика), REST API (JSON), аутентификация с хэшированием паролей.

Особенности: Работа с PostgreSQL через JPA, защита роутов (Auth Guard), полная адаптивность интерфейса.
