Лабораторная работа №1: Веб-стек и FastCGI сервер

Стек: Java (FastCGI SDK), JavaScript (AJAX), HTML5, CSS3, Apache httpd.

Что реализовано:

— Frontend: Статическая HTML-страница с блочной версткой. Использованы продвинутые техники CSS (псевдоэлементы, селекторы дочерних элементов и потомков, каскадирование).

— Валидация: На стороне клиента (JavaScript) реализована проверка входных данных (координат X, Y и радиуса R) перед отправкой запроса.

— Backend: FastCGI-сервер на Java, который обрабатывает GET-запросы, выполняет математический расчет попадания точки в область и возвращает результат.

— Сетевое взаимодействие: Реализован принцип AJAX для обновления таблицы результатов без перезагрузки всей страницы.

— Инфраструктура: Настройка и запуск веб-сервера Apache httpd на удаленном сервере (Helios) с проксированием запросов к FastCGI-приложению.
