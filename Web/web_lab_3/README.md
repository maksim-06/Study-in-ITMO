Лабораторная работа №3: JavaServer Faces (JSF) и ORM (JPA)

Стек: Java Enterprise (JSF, PrimeFaces), JPA (EclipseLink/Hibernate), БД (Postgres), WildFly.

Что реализовано:

— Компонентный UI: Интерфейс построен на базе Facelets-шаблонов. Использована библиотека PrimeFaces (компоненты p:datatable, p:clock и др.) для создания современного и отзывчивого UI.

— ORM-уровень: Внедрена работа с базами данных через JPA (Java Persistence API). Вместо написания голых SQL-запросов использован объектно-реляционный маппинг.

— Managed Beans: Реализована серверная логика на управляемых бинах с различными областями видимости (SessionScoped, ApplicationScoped).

— Интерактивный график: Динамическая отрисовка области. Реализован механизм пересчета координат при клике по картинке с мгновенной отправкой данных на сервер через JSF AJAX.

— Навигация: Логика переходов между страницами (стартовая и основная) вынесена в отдельный конфигурационный файл (Navigation Rules).
