📂 НАВИГАЦИЯ ПО КУРСАМ И СТРУКТУРЕ РЕПОЗИТОРИЯ

🌐 1. Web (Веб-программирование)

👉 Эволюция от базовых протоколов до современной Fullstack-разработки (SPA) с разделением фронтенда и бэкенда.

💻 2. Programming (Программирование на Java)

👉 Эволюция от базового синтаксиса до создания распределенных многопоточных систем с сетевым взаимодействием и СУБД.

💾 3. Databases (Базы данных)

👉 Фокус: Проектирование реляционных схем данных и нормализация таблиц до уровня BCNF. Оптимизация сложных SQL-запросов, построение индексов, написание триггеров и хранимых процедур на языке PL/pgSQL в СУБД PostgreSQL.

🧠 4. Algorithms and data structures (Алгоритмы и структуры данных)

👉 Фокус: Решение сложных алгоритмических задач с платформы Яндекс.Контест. Реализация эффективных структур данных, разбор пространственной и временной сложности алгоритмов (O(n)) и компиляция с использованием C++20 (Clang 17.0.1).

🛠 5. OPD (Организация ЭВМ / Архитектура БЭВМ)

👉 Фокус: Низкоуровневое программирование на ассемблере, построение Unix-иерархий и организация программного обмена данными с внешними устройствами.

💻 6. Computer architecture (Архитектура компьютерных систем)

👉 Фокус: Изучение устройства вычислительных платформ и устранение "магического мышления" на стыке софта и железа. Программное моделирование систем команд (acc32, f32a, m68k, risc-iv) в симуляторе Wrench, проектирование принципиальных логических схем процессоров и анализ уязвимостей (Rowhammer DRAM).

💎 7. Computational Mathematics / ВычМат (Вычислительная математика)

👉 Фокус: Алгоритмическое решение математических задач, не имеющих аналитического решения. Разработка на Python (NumPy, Matplotlib) программных модулей для решения СЛАУ, нелинейных уравнений, численного интегрирования, аппроксимации (МНК) и интерполяции.

⚙ 8. OPI (Основы программной инженерии)

👉 Фокус: Промышленные стандарты обеспечения качества и жизненного цикла ПО. Управление сложной топологией систем контроля версий (Git, SVN), автоматизация сборки с помощью Apache Ant (15 целей), работа с требованиями (SRS/RUP, UML) и профилирование систем в VisualVM.

🚀 ИНФРАСТРУКТУРА И КАЧЕСТВО КОДА

🔥 Исходные коды всех проектов снабжены подробными разборами логики в подпапках.

🔥 Некоторые программы покрыты тестами и успешно прошли автоматическую валидацию (включая тесты JUnit, библиотеки тестирования и удаленные серверы автоматической проверки).
