📂 НАВИГАЦИЯ ПО КУРСАМ И СТРУКТУРЕ РЕПОЗИТОРИЯ

💎 Computer architecture — Архитектура компьютера
👉 Изучение устройства вычислительных платформ, устранение "магического мышления" на стыке софта и железа. Программное моделирование различных процессорных архитектур (acc32, f32a, m68k, risc-iv) в симуляторе Wrench, проектирование принципиальных логических схем процессоров и аналитические исследования аппаратных уязвимостей.

💎 Computational Mathematics / ВычМат — Вычислительная математика
👉 Алгоритмическое решение математических задач, не имеющих аналитического решения. Программная реализация на Python (NumPy, Matplotlib) методов решения СЛАУ, нелинейных уравнений, численного интегрирования, аппроксимации данных (МНК) и интерполяции.

💎 Software Engineering / OPI — Основы программной инженерии
👉 Промышленные подходы к обеспечению качества и жизненного цикла ПО. Управление сложной топологией систем контроля версий (Git, SVN), автоматизация сборки с помощью Apache Ant, работа с требованиями (SRS/RUP, UML) и профилирование систем в VisualVM.

💎 Databases — Базы данных
👉 Проектирование реляционных схем данных, нормализация таблиц до BCNF, оптимизация сложных SQL-запросов, написание триггеров и хранимых процедур на языке PL/pgSQL в СУБД PostgreSQL.

💎 Web & Programming — Прикладное программирование и Веб-разработка
👉 Создание распределенных приложений и полнофункциональных Fullstack-систем. Работа с платформой Java EE (Сервлеты, дескрипторы развертывания), организация бизнес-логики и интеграция с базами данных.

💎 Algorithms and data structures / OPD — Алгоритмы и низкоуровневые системы
👉 Разработка эффективных алгоритмов, работа со структурами данных (С++20, Clang) и низкоуровневое программирование (ОПД, Ассемблер, БЭВМ, управление прерываниями и архитектура систем ввода-вывода).
