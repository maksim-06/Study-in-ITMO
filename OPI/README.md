🚀 ТЕКУЩИЕ КОМПЕТЕНЦИИ

🗺 ОБЗОР ВЫПОЛНЕННЫХ РАБОТ (ROADMAP)



📑 Лабораторная работа №1: Анализ требований и проектирование (SRS \& UML)

👉 Проект: Разработка спецификации для веб-портала путешественников на базе TravelAsk.

👉 Результат: Создан документ SRS по стандарту RUP, спроектированы Use-Case диаграммы, проведена оценка приоритетов, рисков и трудозатрат.



📑 Лабораторная работа №2: Управление версиями в командной разработке

👉 Задача: Воспроизведение сложной топологии коммитов и веток в системах Git и SVN (граф из 15 узлов) от лица двух участников.

👉 Результат: Отработаны навыки параллельной разработки, синхронизации изменений и ручного разрешения конфликтов слияния (merge conflicts).



📑 Лабораторная работа №3: Автоматизация сборки проектов (Apache Ant)

👉 Задача: Написание модульного сценария автоматизации build.xml для Java-приложения с выносом параметров в .properties файлы.

👉 Результат: Реализовано 15 целей (targets), включая компиляцию, junit-тестирование, валидацию xml, работу с хэш-суммами, деплой по SCP и автоматические коммиты в SVN/Git.



📑 Лабораторная работа №4: Управление и профилирование систем (JMX \& VisualVM)

👉 Задача: Разработка MBean-классов для мониторинга бизнес-метрик и анализ производительности распределенного Java-приложения.

👉 Результат: Локализованы и устранены критические дефекты (блокировки ввода-вывода PrintWriter и избыточная работа движка Rhino) с проверкой результатов в VisualVM.
