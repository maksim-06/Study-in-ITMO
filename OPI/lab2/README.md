🎯 ЦЕЛЬ РАБОТЫ



Освоить консольный инструментарий (CLI) систем Git и SVN, изучить механизмы ветвления, слияния и синхронизации данных, а также получить практические навыки разрешения конфликтов при совместной работе над кодом.



🛠 ЧТО РЕАЛИЗОВАНО В ХОДЕ РАБОТЫ



💎 Конфигурация репозиториев: Развертывание и полная настройка локальных репозиториев SVN и Git в Unix-подобной среде окружения.

💎 Эмуляция командной работы: Пошаговое воспроизведение сложного распределенного графа ревизий, состоящего из 15 независимых узлов.

💎 Ролевая модель: Имитация параллельной деятельности двух виртуальных разработчиков ("синий" и "красный" пользователи) в рамках единого проекта.

💎 Ветвление и навигация: Последовательное создание изолированных веток, коммитов, фиксация изменений и переключение между контрольными состояниями проекта.

💎 Разрешение конфликтов: Обнаружение, локализация и ручное устранение структурных конфликтов слияния (merge conflicts), возникших при одновременной правке одних и тех же строк кода.

💎 Сравнительный анализ: Проведение детального анализа различий в подходах к хранению версий и архитектуре данных между Git и SVN.
