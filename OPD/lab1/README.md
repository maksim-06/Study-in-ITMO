Лабораторная работа №1: Работа с файловыми структурами и потоками данных

Стек: Bash, Linux CLI, I/O Redirection.

Что реализовано:

Управление ссылками:

Созданы символические ссылки (ln -s) для гибкого обращения к файлам и директориям.

Созданы жесткие ссылки (ln) для обеспечения целостности данных (файл azurill4).

Манипуляция данными и рекурсия:

Реализовано рекурсивное копирование директорий (cp -r).

Выполнено объединение содержимого нескольких файлов в один с помощью перенаправления потоков вывода (cat file1 file2 > result).

Сложная фильтрация и поиск:

Использование конвейеров (Pipes) для связки команд: ls, grep, sort, head, tail.

Реализована сортировка данных по различным критериям: дате модификации, количеству жестких ссылок, количеству строк (wc -l).

Настроена обработка стандартных потоков ошибок (2> /dev/null, 2> /tmp/errors).

Безопасное удаление:

Отработаны навыки точечного удаления файлов и символических ссылок по маскам (wildcards, например Copy\_\*).

Удаление непустых директорий и сложных структур.
