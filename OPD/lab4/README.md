Лабораторная работа №4: Анализ программных комплексов и подпрограмм на БЭВМ

Стек: Ассемблер БЭВМ, Машинный код (Hex), Архитектура ЭВМ.

Что реализовано:

Анализ программного комплекса: Выполнено восстановление (декомпиляция) текста основной программы и вызываемой подпрограммы из дампа памяти.

Механизмы вызова функций: Разобрана логика передачи управления между модулями, сохранение адреса возврата и передача параметров через регистры или ячейки памяти.

Комплексная трассировка: Проведена пошаговая трассировка всего комплекса, включая моменты перехода в подпрограмму и возврата в основное тело программы.

Системный анализ:

Определено назначение программного комплекса (реализация алгоритма с использованием модульной структуры).

Сформировано описание области допустимых значений для входных переменных и итогового результата.

Работа с расширенным набором команд: Анализ специфических инструкций управления (коды FXXX, EXXX) и косвенной адресации для работы с динамическими адресами.
