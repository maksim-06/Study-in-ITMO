Основы профессиональной деятельности (Архитектура ЭВМ)

В этом разделе собраны лабораторные работы, посвященные изучению низкоуровневого программирования, архитектуры ЭВМ и администрирования Unix-подобных систем.

🚀 Ключевые навыки:

Linux CLI: Свободное владение командной строкой, управление правами доступа (chmod), работа с потоками ввода-вывода и конвейерами (Pipes).

Low-level Programming: Написание и оптимизация программ на языке ассемблера для базовой ЭВМ.

Computer Architecture: Понимание работы процессора (регистры, АЛУ, шины данных), циклов исполнения команд и организации памяти.

System Programming: Реализация драйверов асинхронного ввода-вывода и обработчиков прерываний в режиме реального времени.

📂 Обзор работ:

Лабораторная работа №1: Linux CLI и права доступа

Суть: Построение сложной иерархии файлов и настройка прав доступа.

Инструменты: bash, chmod, grep, sort, find. Работа с регулярными выражениями и фильтрацией данных.

Лабораторная работа №2-4: Анализ и оптимизация машинного кода

Суть: Реверс-инжиниринг программ на уровне hex-кодов, трассировка алгоритмов и оптимизация команд по тактам.

Инструменты: Ассемблер БЭВМ, эмулятор BComp.

Лабораторная работа №5: Асинхронный ввод-вывод

Суть: Разработка драйвера для взаимодействия с внешними устройствами (ВУ-3, ВУ-7) с использованием протокола подтверждения готовности (polling).

Особенности: Работа с кодировками (Windows-1251) и склейка 8-битных данных в 16-битные регистры.

Лабораторная работа №6: Система прерываний

Суть: Программирование асинхронной обработки событий от внешних устройств.

Особенности: Настройка векторов прерываний, маскирование сигналов и реализация ОДЗ на уровне процессора.
