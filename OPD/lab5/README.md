Лабораторная работа №5: Асинхронный ввод-вывод и работа с периферией

Стек: Ассемблер БЭВМ, Внешние устройства (ВУ-3), Кодировка Windows-1251.

Что реализовано:

Драйвер асинхронного вывода: Разработана программа для передачи данных на внешнее устройство (ВУ-3) по принципу асинхронного обмена (handshaking).

Алгоритм опроса готовности: Реализован программный опрос флага готовности ВУ (polling). Процессор ожидает сигнала от устройства (кнопка готовности) перед отправкой следующего байта.

Работа с памятью и строками:

Реализована нестандартная упаковка строк: по два символа (байта) в одну 16-битную ячейку памяти.

Организован доступ к данным по адресу 608 (в десятичной системе) с использованием смещений.

Кодирование данных: Сообщение «Ключ» представлено в кодировке Windows-1251 (HEX: CA, CB, DE, D7).

Логика завершения: Программа корректно обрабатывает стоп-символ 00 (NUL) для остановки процесса вывода.



Дополнение: Сравнение беззнаковых чисел и работа с ВУ-7

Задача: Реализация ввода двух 16-разрядных беззнаковых чисел и вывод максимума на семисегментный индикатор.

Что реализовано:

Составной ввод данных: Реализован алгоритм считывания 16-битного числа с 8-битного порта ВУ-3 в два этапа (сначала старший байт, затем младший) с последующей склейкой в аккумуляторе.

Беззнаковая арифметика: Настроена логика сравнения чисел с использованием флагов процессора, исключающая ошибки интерпретации знакового бита.

Интерфейс вывода: Организован вывод результата на ВУ-7 (семисегментный индикатор) для визуализации наибольшего из двух введенных значений.

Синхронизация: Программа ожидает готовности ВУ перед каждым этапом ввода, реализуя надежный протокол обмена данными.
