Лабораторная работа №2: Анализ и оптимизация машинного кода БЭВМ

Стек: Ассемблер БЭВМ, Машинный код (Hex), Архитектура фон Неймана.

Что реализовано:

Реверс-инжиниринг: Проведен анализ последовательности машинных команд (hex-кодов) для определения заложенной математической функции.

Арифметико-логический анализ:

Выполнена трассировка программы: пошаговое отслеживание изменений в аккумуляторе (AC) и регистрах.

Определены области представления и допустимых значений для знаковых чисел и логических операций (16-битные наборы).

Оптимизация: Разработан альтернативный вариант программы с использованием меньшего количества команд при сохранении идентичного результата (минимизация тактов процессора).

Работа с памятью: Изучены механизмы прямой и косвенной адресации, используемые в наборе команд (коды 0XXX, AXXX, EXXX и др.).
