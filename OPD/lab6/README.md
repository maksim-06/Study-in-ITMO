Лабораторная работа №6: Программирование в режиме прерываний

Стек: Ассемблер БЭВМ, Векторные прерывания, Периферийные устройства (ВУ-2, ВУ-3).

Что реализовано:

— Многозадачность на уровне CPU: Реализовано разделение на фоновую программу (инкремент переменной X в цикле) и асинхронные обработчики прерываний от внешних устройств.

— Обработка прерываний:

\* ВУ-3: По сигналу готовности выполняется расчет функции f(x) = 3x-1 с последующим выводом результата на устройство.

\* ВУ-2: По сигналу готовности меняется знак содержимого регистра данных (РД) и результат записывается обратно в ячейку X.

— Управление состоянием (ОДЗ): Внедрена жесткая проверка области допустимых значений (ОДЗ). При выходе переменной X за границы выполняется автоматический сброс в минимально допустимое значение.

— Системное администрирование: Настройка вектора прерываний, маскирование неиспользуемых сигналов и корректное сохранение/восстановление контекста процессора.
