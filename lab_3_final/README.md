# 🚀 Многопоточный HTTP File Server

Простой многопоточный HTTP-сервер для раздачи статических файлов на C#.

## 📋 Функциональность

- ✅ Многопоточная обработка запросов
- ✅ Раздача статических файлов (HTML, CSS, JS, изображения, документы)
- ✅ Автоматическое определение MIME-типов
- ✅ Защита от path traversal атак
- ✅ Очередь запросов

## 🛠 Сборка проекта

### Требования
- .NET Framework 3.5 или выше ✅
- Rider IDE

### Сборка через Rider
1. Открой проект в Rider
2. Build → Rebuild Solution
3. Запусти с аргументами: `8080 C:\WebServer 5` (это как пример аргументов)


## **СПОСОБ ЗАПУСКА:**

### **Через Rider:**
