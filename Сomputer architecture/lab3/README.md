💎 ИССЛЕДУЕМЫЕ АЛГОРИТМЫ ПО ВАРИАНТУ



1️⃣ REVERSE\_STRING\_PSTR (Реверс строки Pascal) — Архитектура acc32

👉 Суть: Посимвольное чтение строки до символа '\\n' через порт 0x80, её разворот внутри 32-байтного буфера, заполнение остатка символами '\_' и выдача результата в порт 0x84. При переполнении возвращается 0xCCCCCCCC.



2️⃣ LITTLE\_TO\_BIG\_ENDIAN (Конвертация байт-ордера) — Архитектура f32a

👉 Суть: Побайтовое разбиение 32-битного машинного слова с помощью маскирования (0xFF) и логических сдвигов, перенос байт во временный стек возвратов (R-stack) и сборка числа в обратном порядке (из Little-Endian в Big-Endian).



3️⃣ STACK\_BASED\_CALCULATOR (Стековый калькулятор RPN) — Архитектура m68k

👉 Суть: Вычисление арифметических выражений, заданных в постфиксной форме с пробельными разделителями. Реализован парсинг токенов, математические операции (+, -, \*, /) над 32-битными числами, детекция деления на ноль (-1) и переполнения разрядной сетки (0xCCCCCCCC).



4️⃣ FIBONACCI (Числа Фибоначчи) — Архитектура risc-iv-32

👉 Суть: Вычисление N-го члена последовательности Фибоначчи для целого неотрицательного числа, поданного в ячейку 0x80. Реализована обработка отрицательных аргументов (-1) и переполнения знаковой 32-битной сетки (0xCCCCCCCC).



📊 СПЕЦИФИКА РЕАЛИЗАЦИИ ISA И ОБРАБОТКИ ДАННЫХ

🔹 acc32: Вся логика завязана на единственный регистр-аккумулятор, работа со строками реализована через циклы и косвенную адресацию ячеек памяти.

🔹 f32a: Использована процедурная абстракция (выделены подпрограммы сдвигов). Обмен данными и управление контекстом производятся через стек данных и стек возвратов (>r, r>).

🔹 m68k: Задействованы развитые режимы адресации CISC (с преддекрементом и постинкрементом регистров-указателей) для эффективного парсинга строки, реализована структура вложенных подпрограмм.

🔹 risc-iv-32: Архитектура кода использует фиксированный набор команд и работу с процедурным стеком (сохранение адресов возврата) для реализации вложенных вызовов и рекурсивных спусков.

🔹 Принципиальная схема: Спроектирована графическая принципиальная схема процессора для m68k на уровне логических узлов.
