# Курс: Алгоритмы и структуры данных 🧠🚀

Репозиторий содержит решения алгоритмических задач с платформы **Яндекс.Контест** и их детальные аналитические разборы.

## 📌 О проекте

Каждое решение сопровождается подробным отчетом с разбором логики, обоснованием выбора структур данных и математическим расчетом сложности алгоритма.

* **Платформа задач:** Яндекс.Контест
* **Формат разборов:** Документы в форматах `.pdf` и `.docx`

## 🛠 Технологический стек

* **Язык программирования:** C++ (стандарт C++20)
* **Компилятор:** Clang 17.0.1
* **Инструменты автоматизации:** Git
