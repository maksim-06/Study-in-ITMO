# Телефонная книга

Программа для управления контактами с использованием бинарного дерева поиска (BST).

## Сборка и запуск

### Требования
- CMake 3.10 или выше
- Компилятор C (GCC, Clang, MSVC)

### Сборка проекта

```bash
# Клонируйте репозиторий
git clone <ваш-url-репозитория>
cd lab1

# Создайте папку для сборки
mkdir build
cd build

# Соберите проект
cmake ..
make
