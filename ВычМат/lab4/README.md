Лабораторная работа №4: Аппроксимация функции методом наименьших квадратов

Стек: Python (NumPy, Matplotlib).

Что реализовано:

Метод наименьших квадратов (МНК): Реализован алгоритм подбора оптимальных коэффициентов для аппроксимации табличных данных.

Исследование различных моделей: Программа анализирует и строит аппроксимации для следующих типов функций:

Линейная;

Полиномиальная (2-й и 3-й степени);

Экспоненциальная;

Логарифмическая;

Степенная.

Статистический анализ:

Расчет среднеквадратического отклонения для каждой модели.

Вычисление коэффициента корреляции Пирсона для линейной зависимости.

Оценка качества аппроксимации через коэффициент детерминации.

Визуализация: Построение графиков исходной функции и всех полученных аппроксимаций на одном холсте для наглядного сравнения.
