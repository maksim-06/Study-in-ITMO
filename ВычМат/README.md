🧮 КУРС: ВЫЧИСЛИТЕЛЬНАЯ МАТЕМАТИКА (COMPUTATIONAL MATHEMATICS) 🧮



Данный раздел репозитория содержит комплексную реализацию фундаментальных численных методов на языке Python. Проект посвящен разработке и анализу алгоритмов для решения математических задач, не имеющих аналитического решения, с жестким акцентом на точность, сходимость и вычислительную сложность.



🎯 КЛЮЧЕВЫЕ КОМПЕТЕНЦИИ



💎 Numerical Analysis: Реализация методов решения СЛАУ, нелинейных уравнений и систем.

💎 Data Modeling: Аппроксимация и интерполяция данных с использованием статистических метрик (R-square, коэффициент Пирсона).

💎 Python for Science: Активное использование библиотек NumPy для матричных вычислений и Matplotlib для визуализации данных.

💎 Validation: Автоматическая проверка условий сходимости (например, диагональное преобладание) и оценка погрешностей (правило Рунге).



🛠 ТЕХНОЛОГИЧЕСКИЙ СТЕК



🐍 Язык программирования: Python 3.10+

📊 Библиотеки: NumPy, Matplotlib



🗺 ОБЗОР ВЫПОЛНЕННЫХ РАБОТ (ROADMAP)



📑 Лабораторная работа №1: Решение СЛАУ

👉 Суть: Итерационные методы решения линейных систем. Автоматическая перестановка строк для достижения диагонального преобладания.

👉 Методы: Метод простой итерации, Метод Гаусса-Зейделя.



📑 Лабораторная работа №2: Нелинейные уравнения и системы

👉 Суть: Поиск корней трансцендентных уравнений и решение систем нелинейных уравнений.

👉 Методы: Метод половинного деления, Метод хорд, Метод Ньютона, Метод секущих. Графическое отделение корней.



📑 Лабораторная работа №3: Численное интегрирование

👉 Суть: Вычисление определенных интегралов с автоматическим подбором шага сетки.

👉 Методы: Метод прямоугольников (левые, правые, средние), Метод трапеций, Метод Симпсона. Оценка погрешности по правилу Рунге.



📑 Лабораторная работа №4: Аппроксимация функций (МНК)

👉 Суть: Нахождение наилучшего эмпирического приближения табличной функции методом наименьших квадратов.

👉 Методы: Линейная, полиномиальная, экспоненциальная и логарифмическая модели. Анализ качества через R-square и коэффициент корреляции.



📑 Лабораторная работа №5: Интерполяция функций

👉 Суть: Поиск приближенных значений функции при заданных значениях аргумента, отличных от базовых узловых точек.

👉 Методы: Многочлен Лагранжа, Формулы Ньютона (вперед/назад), Формула Гаусса, Формулы Стирлинга и Бесселя.



📑 Лабораторная работа №6: Численное решение обыкновенных дифференциальных уравнений

👉 Суть: Решение задачи Коши для ОДУ с контролем точности шага интегрирования.

👉 Методы: Усовершенствованный метод Эйлера, Метод Рунге-Кутта 4-го порядка, Многошаговый метод Адамса (Предиктор-Корректор).
