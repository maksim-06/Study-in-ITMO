🎯 ЦЕЛЬ РАБОТЫ



Успешно решить задачу интерполяции — то есть найти приближённые значения функции при заданных значениях аргумента, которые не совпадают с исходными узловыми точками.



📊 ИСХОДНЫЕ ДАННЫЕ (ВАРИАНТ 8)



📍 Узлы сетки (x): значения от 1.10 до 2.00 с постоянным шагом h = 0.15.

🎯 Вычисляемые точки интерполяции:

🔹 X1 = 1.852 — находится ближе к концу таблицы, поэтому для расчёта применяется формула Ньютона для интерполирования назад.

🔹 X2 = 1.652 — находится ближе к центру таблицы, поэтому применяется первая интерполяционная формула Гаусса.



🛠 РЕАЛИЗОВАННЫЕ МЕТОДЫ



Разработанный программный комплекс поддерживает расчёт сразу 5 методами:

💎 Многочлен Лагранжа (общий случай)

💎 Формулы Ньютона (для интерполирования вперёд и назад)

💎 Первая формула Гаусса (центральные разности)

💎 Формула Стирлинга

💎 Формула Бесселя



💻 ТЕХНОЛОГИЧЕСКИЙ СТЕК



🐍 Язык программирования: Python 3.10+

📊 Библиотеки визуализации: Matplotlib



🚀 Доступные режимы ввода данных в программе:

👉 Способ 1 — Ручной ввод узлов через консоль.

👉 Способ 2 — Чтение данных из текстового файла. Если файл повреждён или отсутствует, скрипт автоматически загрузит стандартные данные Варианта 8.

👉 Способ 3 — Автоматическая генерация узлов по выбранной математической функции: sin(x), exp(x) или x².



📈 РЕЗУЛЬТАТЫ И ВЫВОДЫ



🔥 После завершения расчётов программа автоматически выводит в консоль красивую таблицу конечных разностей и строит интерактивный график.

🔥 На графике наглядно сопоставляются результаты всех методов интерполяции, исходные узлы и искомая целевая точка.



💡 Главный вывод: Внутри отрезка интерполяции абсолютно все реализованные методы показывают высочайшую сходимость результатов с точностью от 0.0001 до 0.000001! При этом Ньютон идеален для краёв таблицы, а Гаусс — для её центра.
