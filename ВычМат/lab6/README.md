🎯 ЦЕЛЬ РАБОТЫ



Научиться численно решать задачу Коши для обыкновенных дифференциальных уравнений, находить значения искомой функции на заданном интервале и оценивать погрешность вычислений.



📊 РЕАЛИЗОВАННЫЕ ДЛЯ ТЕСТИРОВАНИЯ УРАВНЕНИЯ



Программа поддерживает интегрирование следующих типов ОДУ:

1️⃣ y' = y \* cos(x)

2️⃣ y' = x + y

3️⃣ y' = -2xy



🛠 РЕАЛИЗОВАННЫЕ МЕТОДЫ



Разработанный программный комплекс выполняет расчёты тремя методами:

💎 Усовершенствованный метод Эйлера (2-го порядка точности)

💎 Метод Рунге-Кутта (4-го порядка точности)

💎 Метод Адамса (многошаговый метод: Предиктор-Корректор)

💎 Оценка погрешности: реализовано правило Рунге для одношаговых методов



💻 ТЕХНОЛОГИЧЕСКИЙ СТЕК



🐍 Язык программирования: Python 3.10+

📊 Библиотеки визуализации: Matplotlib



🚀 ДОСТУПНЫЕ РЕЖИМЫ И НАСТРОЙКИ В ПРОГРАММЕ



👉 Выбор любого из 3 доступных дифференциальных уравнений.

👉 Гибкая настройка начальных условий (x0, y0) и конечной точки интервала (xn).

👉 Индивидуальное задание шага интегрирования (h) и требуемой точности (e).



📈 РЕЗУЛЬТАТЫ И ВЫВОДЫ



🔥 После завершения расчётов программа автоматически выводит в консоль красивую таблицу результатов для последних 5 точек интервала с точными, модифицированными и приближёнными значениями.

🔥 Строится интерактивный график, наглядно сопоставляющий графики точного решения и всех численных методов.



💡 Главный вывод: Метод Рунге-Кутта 4-го порядка показал наивысшую точность среди одношаговых методов. Многошаговый метод Адамса также демонстрирует отличную сходимость, а усовершенствованный метод Эйлера подходит для быстрой оценки поведения функции.
