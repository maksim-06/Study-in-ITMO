Лабораторная работа №1: Решение систем линейных алгебраических уравнений (СЛАУ)

Стек: Python, Численные методы.

Что реализовано:

Программная реализация численного метода: Алгоритм вынесен в отдельный компонент (класс/метод), принимающий параметры системы.

Гибкий ввод данных: Поддержка ввода коэффициентов матрицы как с клавиатуры, так и из файла (формат матрицы до 



).

Функционал для прямых/итерационных методов:

Прямые методы: Вычисление определителя через треугольную матрицу, вывод вектора неизвестных и вектора невязок.

Итерационные методы: Проверка и достижение диагонального преобладания (перестановка строк/столбцов), расчет нормы матрицы и мониторинг погрешности на каждой итерации.

Валидация и сравнение: Сопоставление полученных результатов с результатами готовых математических библиотек для оценки точности собственной реализации.
