Лабораторная работа №4: Оптимизация SQL-запросов и анализ производительности

Стек: PostgreSQL, Query Optimization, Indexing, EXPLAIN ANALYZE.

Что реализовано:

— Анализ планов выполнения: Изучение стратегий выполнения запросов (EXPLAIN ANALYZE) до и после оптимизации. Сравнение различных типов сканирования (Seq Scan vs. Index Scan) и соединений (Nested Loop, Hash Join).

— Индексация:

\* Проектирование и внедрение индексов (B-tree) для ускорения фильтрации и соединений.

\* Обоснование выбора атрибутов для индексации на основе селективности данных.

— Оптимизация соединений: Сравнительный анализ планов для LEFT JOIN при работе с таблицами учебного процесса (Н\_ЛЮДИ, Н\_ВЕДОМОСТИ и др.).

— Оценка стоимости (Cost): Прогнозирование и расчет стоимости выполнения запросов, выбор оптимального плана на основе минимизации дисковых операций ввода-вывода (I/O).
