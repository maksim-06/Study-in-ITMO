Лабораторная работа №3: Нормализация и программирование в PostgreSQL

Стек: SQL, PL/pgSQL, Теория реляционных БД (Нормальные формы).

Что реализовано:

— Нормализация данных:

\* Выявлено минимальное множество функциональных зависимостей для предметной области (сцена с Курноу).

\* Проведен последовательный процесс нормализации схемы до 3NF (Третьей нормальной формы) и BCNF (Нормальной формы Бойса-Кодда).

\* Обосновано отсутствие аномалий вставки, обновления и удаления в итоговой схеме.

— Денормализация: Проведен анализ производительности и описаны случаи, в которых денормализация данных была бы полезна для оптимизации чтения.

— PL/pgSQL Программирование:

\* Триггерная функция: Разработана логика на языке PL/pgSQL для автоматической проверки бизнес-правил.

\* Триггер: Реализована автоматизация вызова функции при операциях INSERT или UPDATE для обеспечения целостности данных.
