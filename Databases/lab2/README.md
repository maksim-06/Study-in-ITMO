Лабораторная работа №2: Сложные SQL-запросы и агрегация данных

Стек: PostgreSQL, SQL (DML), Joins, Subqueries, Group By.

Что реализовано:

— Работа с соединениями: Использование различных типов соединений (INNER JOIN, LEFT JOIN) для объединения данных из таблиц студентов, ведомостей и учебных планов.

— Фильтрация и условия: Построение сложных условий выборки с использованием логических операторов (AND/OR) и операторов сравнения для анализа успеваемости.

— Агрегатные функции:

\* Расчет количества студентов (COUNT) по группам и возрастам.

\* Вычисление средних показателей (AVG) для сравнительного анализа успеваемости между группами.

— Вложенные запросы (Subqueries): Реализация логики сравнения средней оценки конкретного студента со средним баллом другой группы.

— Группировка и фильтрация групп: Использование GROUP BY и HAVING для поиска учебных планов, по которым обучается более заданного количества групп.

— Работа с датами и приказами: Формирование списков зачисленных студентов на конкретные даты с фильтрацией по курсам и формам обучения.
