Лабораторная работа №2: Объектно-ориентированное программирование

Стек: Java Core (ООП), работа с внешними библиотеками (JAR).

Что реализовано:

— Анализ документации: Работа с Javadoc для изучения структуры базовых классов и правил взаимодействия с ними.

— Наследование: Проектирование иерархии классов и цепочек эволюции объектов в соответствии с принципами ООП.

— Полиморфизм: Переопределение поведения стандартных методов и адаптация логики атак под конкретные виды объектов.

— Инкапсуляция: Управление внутренним состоянием объектов (базовые характеристики и типы) через конструкторы и модификаторы доступа.

— Моделирование логики: Реализация различных механик взаимодействия между объектами (типы атак, расчет урона, изменение статусов).

— Интеграция стороннего кода: Подключение внешней библиотек.
