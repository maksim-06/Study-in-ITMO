Лабораторная работа №3-4: Углубленное ООП и проектирование моделей

Стек: Java Core, SOLID, Exception Handling, Collections.

Что реализовано:

— Архитектурное проектирование: Построение объектной модели предметной области с использованием UML-диаграмм. Реализована иерархия классов с соблюдением принципов SOLID.

— Применение ООП: Практическое использование абстрактных классов, интерфейсов и полиморфизма для эмуляции поведения объектов.

— Современные стандарты Java: Использование перечислений (enum) и записей (record) для моделирования данных.

— Обработка исключений: Разработка собственной иерархии исключений (Checked и Unchecked). Корректная обработка ошибок с переопределением метода getMessage().

— Работа с коллекциями: Хранение и управление группами объектов с помощью ArrayList.

— Контракты объектов: Переопределение базовых методов equals(), hashCode() и toString() в соответствии с правилами языка Java.

— Сценарное моделирование: Реализация вариативного сценария взаимодействия объектов с поддержкой элементов случайности.
