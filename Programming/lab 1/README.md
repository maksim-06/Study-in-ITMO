Лабораторная работа №1: Основы Java и работа с массивами

Стек: Java Core, Math API, CLI.

Что реализовано:

— Обработка данных: Работа с одномерными и двумерными массивами, использование вложенных циклов.

— Математические вычисления: Реализация сложных формул с использованием библиотеки java.lang.Math (тригонометрия, степени, логарифмы, корни).

— Случайные числа: Генерация данных с помощью Math.random в заданном диапазоне.

— Структура кода: Декомпозиция логики — вычисления и форматированный вывод матрицы вынесены в отдельные статические методы.

— CLI и сборка: Опыт ручной сборки исполняемого JAR-архива через консоль.
