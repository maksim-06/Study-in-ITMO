Лабораторная работа №5: Управление коллекциями и интерактивный CLI

Стек: Java Core, Collections (Stack), File I/O, Валидация данных.

Что реализовано:

Интерактивный CLI: Разработан консольный интерфейс с поддержкой 16 команд для управления коллекцией объектов в реальном времени.

Сложная модель данных: Работа с вложенными объектами (Product, Organization, Coordinates, Address) и перечислениями (Enum).

Валидация и парсинг:

Реализован строгий контроль входных данных на соответствие бизнес-правилам (непустые строки, диапазоны чисел, уникальность ID).

Автоматическая генерация системных полей (id, creationDate).

Поддержка ввода сложных объектов через пошаговые приглашения с обработкой ошибок пользователя.

Файловый ввод-вывод: Реализован импорт/экспорт данных в формате CSV. Запись оптимизирована с использованием BufferedOutputStream, чтение — через Scanner.

Автоматизация (Scripting): Поддержка исполнения команд из файлов (execute\_script) с защитой от бесконечной рекурсии.
